📊 1. Dashboard Divisi Umum PTPN IV

Aplikasi web berbasis HTML/CSS/JavaScript yang berfungsi sebagai dashboard rekap perjalanan dinas untuk Divisi Umum PT Perkebunan Nusantara IV. Aplikasi ini memungkinkan pengguna untuk menginput data perjalanan dinas secara manual maupun melalui upload file Excel, dengan auto-calculation pajak yang menyesuaikan berdasarkan vendor.

HTML + CSS
Struktur & Tampilan
JavaScript
Logika Aplikasi
Tailwind CSS
Design System via CDN

Auto-Calculation per Vendor

Vendor Rumus Utama
US DPP = 11/12 x Service Fee | PPN = SF x 11% | PPH23 = SF x 2% | Total = Tagihan + SF + PPN - PPH23
FES PPH23 = 2% x (SF x Qty) | Total = (Tagihan x Qty) + (SF x Qty) | Dibayarkan = Total - PPH23
Lainnya Semua field diinput manual oleh pengguna

📦 6. Sistem Informasi Repository Arsip PTPN4

Aplikasi web full-stack berbasis Laravel 12 yang berfungsi sebagai sistem informasi repository arsip untuk PTPN IV. Aplikasi ini mengelola siklus hidup arsip digital dari mulai pengajuan box, pengisian data, pengiriman, hingga peminjaman arsip.

Laravel 12
Backend Framework
MySQL
Database Server
Tailwind CSS
Frontend Styling

Tiga Level Pengguna

Role Akses Contoh User
Super Admin Akses penuh: manajemen user, divisi, approve, shipping, laporan superadmin
Administrator Approve pengajuan & peminjaman, manajemen divisi, pengiriman, laporan adminumum
User Divisi Buat pengajuan, isi data box, ajukan peminjaman, lihat laporan (divisi sendiri) userkeuangan

Database (9 Tabel)

Tabel Deskripsi
usersAkun pengguna dengan 3 role
divisi24 divisi PTPN IV
pengajuanPengajuan box arsip dengan nomor otomatis (PA-YYYYMM-NNNN)
boxData box arsip dengan nomor box dan RFID
bantexData bantex/stopmap dalam box
dokumenData dokumen dalam bantex dengan file upload
pengirimanData pengiriman box ke repository
history_pengirimanRiwayat perubahan status pengiriman
peminjamanPeminjaman box arsip oleh divisi
